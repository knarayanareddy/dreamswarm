# First Contribution Guide 🐝
Welcome! This guide is for first-time contributors to DreamSwarm.

## Finding an Issue
- Look for [issues labeled `good first issue`](https://github.com/dreamswarm/dreamswarm/labels/good%20first%20issue).
- These are specifically chosen for their narrow scope and available mentorship.

## Your First PR
1. **Fork the repo** and clone it locally.
2. **Setup your environment**: Run `./scripts/dev-setup.sh`.
3. **Create a branch**: `git checkout -b fix/my-first-fix`.
4. **Make changes**: Even a one-line typo fix is a great way to start!
5. **Test**: Run `make test-unit`.
6. **Commit**: Use a [Conventional Commit](commit-conventions.md) (e.g., `fix: correct typo in README`).
7. **Submit**: Push to your fork and open a PR.

## After Submission
- A maintainer will triage your PR within 24 hours.
- We'll provide friendly feedback or approve it for merge!

Need help? Join our Discord or open a Discussion.
